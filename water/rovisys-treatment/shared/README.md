# Shared Water Treatment Assets

This directory contains assets that are shared across the P1 onboarding demo and the full P1-P6 water treatment implementation.

## Structure

- `tag-contracts/` - Tag contract definitions for PLC I/O
- `mappings/` - Controller-to-simulator mappings
- `signal-models/` - Signal generation and physics models
- `diagrams/` - P&ID diagrams and architecture visuals

## Usage

Assets in this directory should be implementation-agnostic where possible. Specific Rockwell or OpenPLC configurations belong in the respective implementation directories.
